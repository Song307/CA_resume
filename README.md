# CA_resume - My Personal Resume Website

이 프로젝트는 Cursor AI를 활용한 '바이브코딩' 강의를 통해 제작된 자기소개(Resume) 웹사이트입니다.

## 배포 주소
기존의 정적 페이지를 GitHub Pages를 통해 배포하였습니다.
- **URL**: [https://song307.github.io/CA_resume/](https://song307.github.io/CA_resume/)

## 프로젝트 소개
- **강의 출처**: Cursor AI를 활용한 바이브코딩
- **목표**: AI 코딩 도구를 활용하여 현대적이고 깔끔한 디자인의 개인 포트폴리오 및 자기소개 페이지 구축
- **주요 내용**: 홈, 이력서, 프로젝트 목록, 연락처 페이지로 구성된 다목적 웹사이트

## 사용 기술 및 기능
- **Frontend**: HTML5, CSS3, JavaScript
- **Framework**: Bootstrap 5 (Start Bootstrap Theme 기반)
- **AI Tools**: Cursor AI (코드 생성 및 최적화 보조)
- **주요 기능**:
  - **Responsive Design**: 모바일, 태블릿, 데스크탑 등 다양한 기기에서 최적화된 화면 제공
  - **Formspree Integration**: 서버 구축 없이 `Contact` 페이지에서 실제 메일 전송이 가능한 폼 기능 구현 (`contact.html`)
  - **Animated Elements**: 부트스트랩 아이콘 및 CSS 애니메이션을 활용한 동적 UI
  - **Project Showcase**: 완료한 프로젝트들을 시각적으로 깔끔하게 정리 (`projects.html`)

## 폴더 구조
- `index.html`: 메인 홈 페이지
- `resume.html`: 상세 이력 및 학력 사항
- `projects.html`: 프로젝트 포트폴리오 목록
- `contact.html`: 연락처 및 문의 폼
- `css/`: 커스텀 스타일 및 부트스트랩 테마 파일
- `assets/`: 이미지, 파비콘 등 정적 자원
